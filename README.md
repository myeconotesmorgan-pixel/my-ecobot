# MyEcoNotes 團隊留守機器人 (EcoBot) 🤖

這是一隻專為生態調查設計的 P2P 虛擬留守員。它能確保深山調查的資料在隊員交替斷線時，依然能完美進行非同步同步！

### 🚀 一鍵免費部署

請點擊下方按鈕，輸入你們的「團隊名稱」與「專屬密碼」，即可免費架設你們專屬的調查機器人！
